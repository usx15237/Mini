# 电影读书小程序

#### 项目介绍
* 内容：通过对知乎日报,豆瓣电影板块的API抓取，以及页面数据的解析获取最近的电影,图书信息，并进行分类，整理，展示
* 技术：对微信小程序开发框架的学习和运用，以及针对rpx的解析
